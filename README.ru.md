## 🌟 Введение в платформу

**RBT** (Research-Based Tunneling) - это продвинутая платформа туннелирования сервер-сервер, построенная на основе `rstun`. Она разработана для сетей с низким качеством и исследовательских сред, используя современные протоколы, такие как **QUIC**, и оптимизации уровня ядра, такие как **Zero-Copy TCP**.

### 🎯 Основные цели платформы
- ✅ Безопасная и быстрая связь в сетях с высокими потерями
- ✅ Обход систем глубокого анализа пакетов (DPI)
- ✅ Простое и автоматизированное управление сетевой инфраструктурой
- ✅ Мониторинг и аналитика в реальном времени
- ✅ Корпоративный уровень безопасности

## 🎯 Ключевые особенности

### 🔒 Продвинутая безопасность
- **Шифрование AEAD** для маскировки трафика
- **Обход DPI** с использованием передовых техник
- **Автоматические SSL/TLS сертификаты** с автопродлением
- **Система многофакторной аутентификации**

### 🚀 Высокая производительность
- **Zero-Copy TCP** с использованием системного вызова Linux `splice()`
- **BBR контроль перегрузки** для максимальной пропускной способности
- **QUIC для UDP** без блокировки головой очереди
- **Избежание копирования памяти на уровне ядра**

### 🎛️ Продвинутое управление
- **Динамическое переключение портов** на основе криптографической синхронизации TOTP
- **Автоматическая смена маршрутов** при обнаружении блокировки
- **Мониторинг в реальном времени** с умными оповещениями
- **Автоматическое резервное копирование и восстановление**

## 🛠️ Быстрая установка

### 🚀 Рекомендуемый метод - Установка в одну строку

```bash
# Полная установка с интеллектуальным скриптом
curl -fsSL https://raw.githubusercontent.com/EHSANKiNG/RBT/main/install-complete-enhanced.sh | bash

# Или с wget
wget -qO- https://raw.githubusercontent.com/EHSANKiNG/RBT/main/install-complete-enhanced.sh | bash
```

### 📦 Метод ручной установки

```bash
# 1. Клонировать репозиторий
git clone https://github.com/EHSANKiNG/RBT.git
cd RBT

# 2. Установить зависимости
npm install

# 3. Запустить интерактивный установщик
npm run rbt-install
```

### 🔧 Расширенные параметры установки

```bash
# Установка с минимальными зависимостями
bash install-complete-enhanced.sh --minimal

# Установка с максимальной безопасностью
bash install-complete-enhanced.sh --security-hardened

# Установка с полным мониторингом
bash install-complete-enhanced.sh --monitoring-full

# Неинтерактивная установка
bash install-complete-enhanced.sh --non-interactive
```

|--------|--------|--------|
